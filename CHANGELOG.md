# Changelog

All notable changes use a simplified Keep a Changelog format. The project follows semantic
versioning after the first tagged release.

## 0.1.0 - 2026-07-18

### Added

- Local-first PDF workspace with durable document and ingestion state.
- Native text, structured table, rendered visual, and optional OCR evidence extraction.
- Versioned Chroma retrieval with explicit deterministic or OpenAI embeddings.
- Optional OpenAI multimodal visual understanding and grounded answer synthesis.
- Server-validated claims and page-level evidence citations.
- Streamlit Ask and Documents core workflow with progressively disclosed evidence, preparation,
  privacy, and contextual help.
- Eight-page Northstar fixture, golden questions, deterministic tests, CI, and local containers.
- Docker-first newcomer quickstart, extension map, product screenshot, issue forms, pull-request
  template, code of conduct, and public-repository checks.
